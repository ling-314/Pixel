## 快速上手

### 配置要求

#### 建议配置
系统：Windows 8.1  
编译器：MSVC

#### 最低配置
系统：Windows 7  
编译器：MSVC

### 安装教程
在github仓库右侧的release中下载对应版本压缩包，将压缩包中的Pixel文件夹解压到项目目录。include文件夹存放头文件lib文件夹存放x64和x86的dll和lib。
